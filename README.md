проверить версию Node.js и npm
```shell
node -v
```
```shell
npm -v
```

инициализация преокта
```shell
npm init
```
установить Gulp глобально (если его еще нет)
```shell
npm install -g gulp-cli
```
и локально для проекта
```shell
npm install gulp --save-dev
```

для запуска проекта выполнить

```shell
gulp
```
или
```shell
gulp style
```
